The AXIOM OS
Design Language

AXIOM OS is the interface language of the Stellarhold: Diaspora archive platform. It communicates cosmic scale through restraint — a terminal-era intelligence system that has persisted across centuries of interstellar civilization.

Design Philosophy
Tone & Mood
Cold. Precise. Ancient intelligence made legible. The system speaks in teal light against near-void backgrounds — a signal from deep infrastructure.
01
Terminal Precision
All text is uppercase and tracked wide. Borders are sharp — zero radius throughout. Information is data, not decoration.
02
Void Depth
Backgrounds approach absolute dark (#05070F). Surfaces emerge in luminance steps, creating depth without color.
03
Signal Clarity
The primary teal (#4cd7f6) is the only saturated color in the system. It marks interactive elements and live data.
04
Persistent Memory
The system implies age through scan-line overlays, grain textures, and compressed type. This is an archive, not a product.
05
Earned Access
Tier badges, locked states, and access gates are first-class UI patterns. Information density scales with clearance level.
06
Minimal Motion
Transitions are 150–300ms and purposeful. Micro-animations reinforce system state, not aesthetic. The scan beam is the only ambient loop.

WCAG 2.1 / 2.2
Accessibility Compliance

AXIOM OS targets WCAG 2.1 AA compliance across all components. Contrast ratios are calculated against the actual rendered surface — not just the page background. Two violations were found and corrected in this build (.type-spec and ::placeholder were using #3d494c at 2.4:1).

1.4.3 / 1.4.6
Color Contrast
All ratios are live-calculated. AA requires 4.5:1 for normal text, 3:1 for large text (≥18pt or ≥14pt bold). AAA requires 7:1 / 4.5:1.
Pair Name Foreground Background Ratio AA AAA

* Large text = ≥18px regular or ≥14px bold. Large text AA threshold is 3:1; AAA is 4.5:1.

2.4.7 / 2.4.11
Focus Indicators
All interactive elements show a 2px solid #4cd7f6 outline on keyboard focus via :focus-visible. Mouse users never see the ring.

1.4.4
Typography & Size Compliance
WCAG requires text be resizable to 200% without loss of content. The smallest text in this system is 9px — used only for decorative uppercase labels and metadata, never for body content or instructions.

2.3.3
Reduced Motion
All looping animations respect prefers-reduced-motion: reduce. Apply this in your implementation:
@media (prefers-reduced-motion: reduce) {
  /* Kill looping ambient animations */
  .scan-beam, .glow-text, .typewriter-text,
  .pulse-dot, .spin-ring, .skeleton {
    animation: none;
  }
  /* Snap transitions for motion-sensitive users */
  *, *::before, *::after {
    transition-duration: 0.01ms !important;
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
  }
}
