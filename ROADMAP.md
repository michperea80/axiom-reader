# AXIOM Reader Roadmap

## Now
- Current-file Notes Dashboard with jump, edit, delete, and export actions.
- Current-document search with match highlighting and next/previous navigation.
- Per-file review status stored locally with recent file records.
- Conservative TTS pronunciation cleanup for all-caps terms and heading Roman numerals.
- Passage highlights separate from review notes.
- Stronger Markdown export with grouped headings and task checkboxes.
- Local app data backup and restore for IndexedDB content.
- Highlight color labels and highlight-specific export filters.
- Review session summary with total notes, highlights, and unresolved tasks.
- Optional pronunciation dictionary import/export.

## Next
- Per-highlight comments and resolved states.
- Saved export presets for common review workflows.
- Cross-file review dashboard with project-wide counts.

## Later
- OCR support for scanned/image PDFs.
- Advanced TTS modes such as read current section, skip headings, and restart section.
- Cross-file search across saved recent documents.
